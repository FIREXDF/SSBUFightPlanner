Installing Mods

You can install mods by:

  • Dragging and dropping mod files
  • Clicking button
  • Clicking button

Managing Mods

Right-click on a mod to access options like:

  • Rename
  • Toggle Enable/Disable
  • Open Folder

⚠️ Warning ⚠️

Remember that FightPlanner will not be fully compatible with movesets and you may need to install them manually.

FightPlanner is not a mod loader it's a mod manager, you will need arcropolis for running mods.

You can see the official guide on GameBanana made by FIREXDF (the developer of FightPlanner) that is made for emulator.

⚠️ Warning ⚠️

What you can still change after the tutorial:

  • The mods directory
  • Change the theme to light & dark.

Known issues:
